# 📈 US Stock Market Analysis

## Problem Statement
Investors find it difficult to compare performance of multiple US stocks 
in one place. This project analyses 1-year historical data of 5 major 
US stocks to identify trends, returns, and volatility.

## Key Findings
- 🏆 Best Performer: Google (113.62% return in 1 year)
- 📉 Worst Performer: Microsoft (-6.36%)
- 🌊 Most Volatile: Google (11.98% std dev)
- 🪨 Most Stable: Apple (6.41% std dev)
- 📅 Worst month across stocks: Feb 2026 (market-wide dip)

## Tech Stack
- Python (yfinance, Pandas)
- Google Sheets (Dashboard + Charts)
- Git & GitHub

## Files
- stock_analysis.py - Main Python script
- stock_analysis.xlsx - Output data with monthly returns
